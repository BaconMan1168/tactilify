Circuit diagram: symbol style

The elements from Claude will include types like "battery", "resistor", "bulb", "capacitor", "switch". How should the SVG renderer represent them?

A — Labeled shapes

Each element is a rect or circle with its label and value. Simple, reliable for all element types. Unknown types get a generic rectangle. No symbol library needed.

B — Standard schematic symbols

Battery = vertical line pairs, resistor = zigzag, bulb = circle-with-X, capacitor = parallel lines, switch = gap with line. Authentic to real circuit diagrams. More code per symbol type.

Context

Option B is more accessible for students who know circuit notation — they can match what they see on the SVG to their textbook. Option A is safer when Claude returns unusual element types that don't have a defined symbol. We could also combine: B for known types, A fallback for unknown.
