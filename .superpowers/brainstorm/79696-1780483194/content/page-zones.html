Page Zone Layout

Where should exploration instructions live on the A4 tactile page? (All zones are braille-rendered.)

TITLE (braille)
DRAWING AREA
INSTRUCTIONS (braille)
KEY (braille)

A — Instructions after drawing

Title → Drawing → Instructions → Key. Instructions appear between the diagram and the key. Reader explores the diagram first, then reads "how to read it" before the key. BANA-standard order.

TITLE (braille)
INSTRUCTIONS (braille)
DRAWING AREA
KEY (braille)

B — Instructions before drawing

Title → Instructions → Drawing → Key. Reader gets the exploration guide before touching the diagram. Better for novice readers — they know what to do before they start. APH-recommended for complex diagrams.

TITLE (braille)
DRAWING AREA
KEY (braille)
INSTRUCTIONS (braille)

C — Instructions at the very bottom

Title → Drawing → Key → Instructions. Key stays adjacent to diagram (easiest cross-reference while exploring). Instructions become an optional appendix. Works well for experienced readers.
